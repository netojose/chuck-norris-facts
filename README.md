# React Boilerplate

## Used technologies

-   React
-   TypeScript
-   Emotion
-   Dotenv
-   Eslint
-   Enzyme
-   Prettier
-   Cypress
-   Jest

## Avalilable commands

### Type-Checking the repo

```shell
npm run type-check
```

### Type-Checking the repo in --watch mode:

```shell
npm run type-check:watch
```

### Building the `bundle`

```shell
npm run build
```

### Run development mode

```shell
npm start
```

### Building the `bundle` and run server

```shell
npm start:prod
```

### Linting

```shell
npm lint
```

### Run unit tests (jest)

```shell
npm test:unit
```

### Run e2e tests (cypress)

```shell
npm test:e2e
```

### Run e2e tests (cypress) opening browser

```shell
npm test:e2e:open
```
